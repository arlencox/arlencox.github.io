Projects

  • SETr

    SETr is an abstract domain library for analyzing programs that manipulate sets.

  • QUICr

    QUICr is an abstract domain library for analyzing programs that manipulate sets that contain numbers.

  • JSAna

    JSAna is a static analyzer for JavaScript that focuses on analyzing library code in absence of code that uses the library.
